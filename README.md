## ⚙️ Cara Setup

### Prasyarat

Pastikan sistem Anda telah terinstal **Python 3.14** atau yang lebih baru.

### 1. Clone Repositori

bash

git clone <url-repositori-anda>

cd random-seed-42


### 2. Buat Virtual Environment

bash

python -m venv venv


Aktifkan virtual environment:

- **Linux / Mac:**

bash

  source venv/bin/activate


- **Windows:**

bash

  venv\Scripts\activate


### 3. Instal Dependensi

Proyek ini menggunakan pyproject.toml untuk manajemen dependensi. Instal seluruh pustaka yang dibutuhkan dengan perintah:

bash

pip install .
